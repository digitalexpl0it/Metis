# Metis internationalization (i18n)

Metis uses a **hybrid** localization stack:

| Component | Format | API |
|-----------|--------|-----|
| `metis-shell`, `metis-settings` | GNU gettext (`.po` → `.mo`) | `metis_i18n::tr` / `trn` |
| `metis-compositor` | Fluent (`.ftl`) | `metis_i18n::tr_ftl` |

Shared crate: `metis-i18n` (locale resolve, catalog load, formatting helpers).

## Locale selection

1. `~/.config/metis/locale.json` override (`"locale": "es"` or `null` for system)
2. Else `LC_ALL` / `LC_MESSAGES` / `LANG`
3. Settings → **System → Language & region**
4. First-run onboarding step 1 (Language & region)

Live apply writes `locale.json`, rebinds gettext/Fluent (via `LANGUAGE=` + a non-`C` `setlocale`), rebuilds Settings UI, sends shell `reload-locale` and compositor `ReloadLocale`.

**Host locales:** Metis does **not** require `locale-gen es_ES.UTF-8`. Catalogs are selected with `LANGUAGE` while `setlocale` uses the requested locale when available, otherwise `en_US.UTF-8` (never bare `C`, which makes gettext ignore `LANGUAGE`). Install catalogs to `/usr/local/share/metis/locale` via `./run-metis.sh --install-session`.

## Catalog layout

```
assets/locale/
  en/ … es/ fr/ de/ pt/ it/ nl/ pl/ ru/ ja/ zh/ ko/ ar/ he/
    LC_MESSAGES/metis.po   # gettext source (English msgids)
    LC_MESSAGES/metis.mo   # compiled
    compositor/metis.ftl   # Fluent (lock / SSD chrome)
```

English (`en`) is the source language. Other folders are **skeleton** catalogs covering Settings navigation, Language & region, Metis Menu rail/chrome, onboarding, and compositor lock/SSD strings. Untranslated UI (most Settings page bodies, many bar widgets) still falls back to English msgids.

Install path: `/usr/share/metis/locale` (`.deb` / packaging) and
`/usr/local/share/metis/locale` (`./run-metis.sh --install-session`), or
`METIS_LOCALE_DIR`. Runtime probes both share prefixes.

## Contributor workflow

```bash
# After adding tr("…") call sites:
./scripts/i18n-extract.sh

# Compile .po → .mo before install / testing:
./scripts/i18n-compile.sh

# Test Spanish catalogs:
# locale.json → { "locale": "es", "formats_from_locale": true }
# or: LANG=es_ES.UTF-8 ./run-metis.sh --session
```

Fluent compositor strings are edited by hand in `*/compositor/metis.ftl`.

## RTL / fonts

GTK: `Widget::set_default_direction` follows `metis_i18n::is_rtl()`.

Compositor lock text uses `unicode-bidi` visual order + `rustybuzz` shaping probe; prefer Noto Sans / Noto Sans CJK / Noto Sans Arabic packages on the host.

## Formatting

`format_short_date` / `format_short_time` / `format_decimal` use chrono localized patterns keyed off the active locale (`formats_from_locale` in `locale.json`).
